## 📝 学习笔记

- 量化交易相比传统手工交易：速度快、无情绪、可回测、纪律性强
- K线：单个时间周期内 OHLC 价格的可视化
- 基本面：盈利能力、估值、成长性、财务健康
- 技术面：趋势指标、动能指标、量价分析、形态识别
